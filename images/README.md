# Images Directory

## Logo Placement

Please place your logo file in this directory with the name `logo.png`.

### Logo Specifications

- **Aspect Ratio**: Square (1:1)
- **Recommended Size**: 200x200px or larger
- **Format**: PNG (preferred) or JPG
- **Background**: Black with white text (as described)

### File Naming

The website expects the logo to be named:
- `logo.png`

If you use a different name or format, update the reference in `index.html` (line 23):
```html
<img src="images/logo.png" alt="Christopher Carson Music Logo" class="logo-img">
```

### Optimization

For best performance, optimize your logo before uploading:
- Keep file size under 100KB
- Use PNG for transparency or crisp text
- Consider using tools like TinyPNG.com for compression
